SODA Remote Control App — Complete Feature Design
What We're Building
A React Native Android app that connects to SODA running on your PC from anywhere in the world. It mirrors the SODA aesthetic (dark, cyan, HUD-style) and lets you control your laptop remotely using your voice and touch.

How the Remote Connection Works (The Bridge)
This is the most important part to understand first.

text

Your Android Phone                    Your PC (Running SODA)
──────────────────                    ──────────────────────
React Native App                      backend/server.py
     │                                       │
     │         Internet (Tunnel)             │
     └──────────────────────────────────────►│
              ngrok / Cloudflare Tunnel       │
              generates a public URL          │
              e.g: https://xyz.ngrok.io       │
                                        Port 8000
                                      (already running)
The key insight: SODA's backend (server.py) already runs a Socket.IO + FastAPI server on port 8000. It's already built for real-time communication. You just need to expose that port to the internet and connect to it from your phone.

You don't need to build a new backend. You need:

A tunnel that makes port 8000 reachable from outside
An Android app that speaks SODA's existing Socket.IO protocol
A few new socket events added to server.py for mobile-specific needs
The Tunnel System — How "Thousands of Miles Away" Works
Option A: ngrok (Easiest to Start)
text

backend/server.py starts on port 8000
ngrok starts automatically alongside it
ngrok gives you: https://abc123.ngrok.io
You hardcode or share this URL with your phone app
Add this to server.py startup:

Python

# backend/tunnel.py — new file
import subprocess
import requests
import time

def start_tunnel(port=8000):
    """Start ngrok tunnel and return public URL"""
    # Start ngrok process
    proc = subprocess.Popen(
        ['ngrok', 'http', str(port), '--log=stdout'],
        stdout=subprocess.PIPE,
        stderr=subprocess.PIPE
    )
    
    # Wait for ngrok to initialize
    time.sleep(2)
    
    # Ask ngrok's local API for the public URL
    try:
        resp = requests.get('http://localhost:4040/api/tunnels')
        tunnels = resp.json()['tunnels']
        public_url = tunnels[0]['public_url']
        return public_url, proc
    except Exception as e:
        return None, proc
The URL gets saved to a file or displayed in the SODA UI so you can copy it to your phone.

Option B: Cloudflare Tunnel (More Stable, Free, No Account Limit)

cloudflared tunnel --url http://localhost:8000

Option C: Fixed Domain (Best for Long-Term)
If your PC has a static IP or you use a Dynamic DNS service (like DuckDNS), you set up port forwarding on your router once. Then your app always connects to soda.yourdomain.com. No ngrok needed.

What's New in server.py — Mobile Auth & Events
You need to add a mobile authentication layer so random people can't connect to your SODA through the tunnel.

Python

# Add to server.py

MOBILE_SECRET = os.getenv("MOBILE_SECRET", "your-secret-key-here")
mobile_sessions = set()  # Track authenticated mobile connections

@sio.on('mobile_auth')
async def handle_mobile_auth(sid, data):
    """Mobile app sends secret key to authenticate"""
    if data.get('secret') == MOBILE_SECRET:
        mobile_sessions.add(sid)
        await sio.emit('mobile_auth_result', {
            'success': True,
            'soda_running': audio_loop is not None,
            'version': '2.1.0'
        }, to=sid)
    else:
        await sio.emit('mobile_auth_result', {
            'success': False,
            'reason': 'Invalid secret'
        }, to=sid)

@sio.on('mobile_voice_command')
async def handle_mobile_voice(sid, data):
    """Receive voice audio from phone, inject into SODA's audio loop"""
    if sid not in mobile_sessions:
        return
    # data['audio'] = base64 encoded PCM audio chunk
    # data['text'] = optional text command (for text mode)
    if audio_loop:
        await audio_loop.inject_mobile_input(data)

@sio.on('mobile_text_command')  
async def handle_mobile_text(sid, data):
    """Text command from mobile — send to Gemini as if user said it"""
    if sid not in mobile_sessions:
        return
    if audio_loop:
        await audio_loop.inject_text_command(data.get('text', ''))

New File: backend/mobile_bridge.py
This file handles forwarding SODA's events to mobile clients:

Python

# backend/mobile_bridge.py

class MobileBridge:
    """
    Sits between AudioLoop and mobile clients.
    Forwards relevant socket events to authenticated mobile sessions.
    """
    
    def __init__(self, sio, mobile_sessions):
        self.sio = sio
        self.mobile_sessions = mobile_sessions
    
    async def forward_to_mobile(self, event, data):
        """Send event to all authenticated mobile clients"""
        for sid in self.mobile_sessions.copy():
            try:
                await self.sio.emit(event, data, to=sid)
            except Exception:
                self.mobile_sessions.discard(sid)
    
    async def forward_status(self, status):
        """Forward SODA status updates (listening/thinking/speaking/idle)"""
        await self.forward_to_mobile('soda_status', {'status': status})
    
    async def forward_transcription(self, text, is_user=True):
        """Forward what was said / what SODA replied"""
        await self.forward_to_mobile('soda_transcription', {
            'text': text,
            'is_user': is_user
        })
    
    async def forward_tool_result(self, tool_name, result):
        """Forward tool execution results for display on mobile"""
        await self.forward_to_mobile('soda_tool_result', {
            'tool': tool_name,
            'result': result
        })
    
    async def forward_audio(self, pcm_bytes):
        """
        Forward SODA's voice response audio to phone.
        This is how you HEAR SODA on your phone from far away.
        Base64 encode the PCM bytes before sending.
        """
        import base64
        await self.forward_to_mobile('soda_audio', {
            'audio': base64.b64encode(pcm_bytes).decode()
        })

What It Does

// screens/ConnectionScreen.js

import React, { useState, useEffect, useRef } from 'react';
import {
  View, Text, TextInput, TouchableOpacity,
  StyleSheet, Animated, KeyboardAvoidingView
} from 'react-native';
import { LinearGradient } from 'expo-linear-gradient';
import SocketService from '../services/SocketService';
import StorageService from '../services/StorageService';
import { COLORS } from '../constants/colors';

export default function ConnectionScreen({ navigation }) {
  const [url, setUrl] = useState('');
  const [secret, setSecret] = useState('');
  const [status, setStatus] = useState('idle'); 
  // idle | connecting | failed | success
  const [errorMsg, setErrorMsg] = useState('');
  
  // Animation values
  const glowAnim = useRef(new Animated.Value(0)).current;
  const fadeAnim = useRef(new Animated.Value(0)).current;
  
  useEffect(() => {
    // Fade in the screen
    Animated.timing(fadeAnim, {
      toValue: 1, duration: 800,
      useNativeDriver: true
    }).start();
    
    // Load saved connection details
    StorageService.getConnection().then(saved => {
      if (saved) {
        setUrl(saved.url);
        setSecret(saved.secret);
        // Auto-attempt connection if we have saved details
        attemptConnection(saved.url, saved.secret);
      }
    });
  }, []);
  
  // Pulsing glow animation when connecting
  const startGlowPulse = () => {
    Animated.loop(
      Animated.sequence([
        Animated.timing(glowAnim, {
          toValue: 1, duration: 1000, useNativeDriver: true
        }),
        Animated.timing(glowAnim, {
          toValue: 0.3, duration: 1000, useNativeDriver: true
        })
      ])
    ).start();
  };
  
  const attemptConnection = async (targetUrl, targetSecret) => {
    setStatus('connecting');
    startGlowPulse();
    
    try {
      const result = await SocketService.connect(targetUrl, targetSecret);
      
      if (result.success && result.soda_running) {
        // Save these credentials for next time
        await StorageService.saveConnection(targetUrl, targetSecret);
        setStatus('success');
        // Brief pause so user sees the success state, then navigate
        setTimeout(() => navigation.replace('Main'), 600);
        
      } else if (result.success && !result.soda_running) {
        // Connected to the server but SODA AudioLoop isn't running
        setStatus('failed');
        setErrorMsg('Connected to PC but SODA is not running. Start SODA first.');
        
      } else {
        setStatus('failed');
        setErrorMsg('Cannot reach SODA. Check the URL and your connection.');
      }
    } catch (err) {
      setStatus('failed');
      setErrorMsg('Connection timed out. Is SODA running?');
    }
  };
  
  // What the status indicator text shows
  const getStatusText = () => {
    switch(status) {
      case 'connecting': return 'ESTABLISHING CONNECTION...';
      case 'failed':     return 'CONNECTION FAILED';
      case 'success':    return 'SODA ONLINE — LINKING...';
      default:           return 'ENTER SODA CONNECTION DETAILS';
    }
  };
  
  const statusColor = {
    idle:       COLORS.muted,
    connecting: COLORS.accent,    // cyan
    failed:     COLORS.error,     // red
    success:    COLORS.success,   // green
  }[status];
  
  return (
    <LinearGradient
      colors={['#0a0a0f', '#0d0d18', '#0a0a0f']}
      style={styles.container}
    >
      <Animated.View style={[styles.content, { opacity: fadeAnim }]}>
        
        {/* Top section — SODA logo/name */}
        <View style={styles.header}>
          {/* Animated ring around the S logo */}
          <Animated.View style={[
            styles.logoRing,
            {
              opacity: glowAnim,
              // The ring glows cyan when connecting
              borderColor: statusColor
            }
          ]}>
            <Text style={styles.logoText}>S</Text>
          </Animated.View>
          <Text style={styles.title}>SODA REMOTE</Text>
          <Text style={styles.subtitle}>SUPER OPTIMIZED DESIGN ASSISTANT</Text>
        </View>
        
        {/* Status indicator line */}
        <View style={styles.statusRow}>
          <View style={[styles.statusDot, { backgroundColor: statusColor }]} />
          <Text style={[styles.statusText, { color: statusColor }]}>
            {getStatusText()}
          </Text>
        </View>
        
        {/* Error message if failed */}
        {status === 'failed' && (
          <Text style={styles.errorMsg}>{errorMsg}</Text>
        )}
        
        {/* Input form — only show when not connecting/success */}
        {(status === 'idle' || status === 'failed') && (
          <KeyboardAvoidingView behavior="padding" style={styles.form}>
            
            <Text style={styles.label}>SODA SERVER URL</Text>
            <TextInput
              style={styles.input}
              value={url}
              onChangeText={setUrl}
              placeholder="https://xyz.ngrok.io"
              placeholderTextColor={COLORS.muted}
              autoCapitalize="none"
              autoCorrect={false}
              keyboardType="url"
            />
            
            <Text style={styles.label}>SECRET KEY</Text>
            <TextInput
              style={styles.input}
              value={secret}
              onChangeText={setSecret}
              placeholder="your-mobile-secret"
              placeholderTextColor={COLORS.muted}
              secureTextEntry={true}
              autoCapitalize="none"
            />
            
            <TouchableOpacity
              style={styles.connectButton}
              onPress={() => attemptConnection(url, secret)}
              activeOpacity={0.7}
            >
              <Text style={styles.connectButtonText}>CONNECT TO SODA</Text>
            </TouchableOpacity>
            
          </KeyboardAvoidingView>
        )}
        
        {/* Scanning lines animation — purely decorative HUD aesthetic */}
        <View style={styles.scanlineContainer} pointerEvents="none">
          {[...Array(8)].map((_, i) => (
            <View key={i} style={[
              styles.scanline,
              { top: `${10 + i * 12}%`, opacity: 0.03 + i * 0.005 }
            ]} />
          ))}
        </View>
        
      </Animated.View>
    </LinearGradient>
  );
}
Screen 2: Main Screen — The Orb
What It Does
Center screen: the animated SODA orb (same feel as desktop)
Orb reacts to SODA's status (cyan = listening, amber = thinking, etc.)
You hear SODA's voice through your phone speaker
Tap the orb to start talking to SODA
Say "show all options" (or tap the grid icon) → orb fades out → diamond menu flies in
screens/MainScreen.js — Explained
JavaScript

// screens/MainScreen.js

import React, { useState, useEffect, useRef, useCallback } from 'react';
import {
  View, StyleSheet, Text, TouchableOpacity,
  Animated, Dimensions, StatusBar
} from 'react-native';
import * as Haptics from 'expo-haptics';
import SocketService from '../services/SocketService';
import AudioService from '../services/AudioService';
import SodaOrb from '../components/SodaOrb';
import DiamondMenu from '../components/DiamondMenu';
import TranscriptionFeed from '../components/TranscriptionFeed';
import { COLORS } from '../constants/colors';

const { width, height } = Dimensions.get('window');

export default function MainScreen({ navigation }) {
  // SODA state mirroring
  const [sodaStatus, setSodaStatus] = useState('idle');
  // idle | listening | thinking | speaking
  
  const [isRecording, setIsRecording] = useState(false);
  const [transcriptions, setTranscriptions] = useState([]);
  const [showDiamond, setShowDiamond] = useState(false);
  const [isConnected, setIsConnected] = useState(true);
  
  // Animation values
  const orbOpacity = useRef(new Animated.Value(1)).current;
  const orbScale = useRef(new Animated.Value(1)).current;
  const diamondOpacity = useRef(new Animated.Value(0)).current;
  const transcriptionOpacity = useRef(new Animated.Value(1)).current;
  
  useEffect(() => {
    setupSocketListeners();
    return () => {
      SocketService.removeAllMobileListeners();
      AudioService.stopAll();
    };
  }, []);
  
  const setupSocketListeners = () => {
    
    // SODA tells us its current state
    SocketService.on('soda_status', (data) => {
      setSodaStatus(data.status);
    });
    
    // What was said / what SODA replied — add to transcription feed
    SocketService.on('soda_transcription', (data) => {
      setTranscriptions(prev => [
        ...prev.slice(-20), // Keep last 20 entries
        {
          text: data.text,
          isUser: data.is_user,
          id: Date.now()
        }
      ]);
      
      // Check if SODA is suggesting to show options
      // (you say "show all options" → SODA processes it → triggers this)
      if (data.text && detectShowOptionsIntent(data.text)) {
        triggerDiamondMenu();
      }
    });
    
    // Play SODA's voice response through phone speaker
    SocketService.on('soda_audio', (data) => {
      AudioService.playPcmBase64(data.audio);
    });
    
    // Connection dropped
    SocketService.on('disconnect', () => {
      setIsConnected(false);
      navigation.replace('Connection');
    });
  };
  
  // Simple intent detection on transcription text
  // Looks for the "show all options" trigger phrase
  const detectShowOptionsIntent = (text) => {
    const lower = text.toLowerCase();
    return lower.includes('show all options') ||
           lower.includes('show options') ||
           lower.includes('open options') ||
           lower.includes('show menu');
  };
  
  // ─── The core animation: Orb disappears, Diamond appears ───
  const triggerDiamondMenu = useCallback(() => {
    Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Medium);
    
    // Phase 1: Orb shrinks and fades out (200ms)
    Animated.parallel([
      Animated.timing(orbOpacity, {
        toValue: 0, duration: 200,
        useNativeDriver: true
      }),
      Animated.spring(orbScale, {
        toValue: 0, speed: 20, bounciness: 0,
        useNativeDriver: true
      }),
      Animated.timing(transcriptionOpacity, {
        toValue: 0, duration: 150,
        useNativeDriver: true
      })
    ]).start(() => {
      // Phase 2: Show diamond menu, then animate it in
      setShowDiamond(true);
      Animated.spring(diamondOpacity, {
        toValue: 1, speed: 12, bounciness: 6,
        useNativeDriver: true
      }).start();
    });
  }, [orbOpacity, orbScale, diamondOpacity, transcriptionOpacity]);
  
  // ─── Reverse: Diamond collapses, Orb returns ───
  const closeDiamondMenu = useCallback(() => {
    Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light);
    
    Animated.timing(diamondOpacity, {
      toValue: 0, duration: 200,
      useNativeDriver: true
    }).start(() => {
      setShowDiamond(false);
      
      Animated.parallel([
        Animated.spring(orbOpacity, {
          toValue: 1, speed: 10, bounciness: 8,
          useNativeDriver: true
        }),
        Animated.spring(orbScale, {
          toValue: 1, speed: 10, bounciness: 10,
          useNativeDriver: true
        }),
        Animated.timing(transcriptionOpacity, {
          toValue: 1, duration: 300,
          useNativeDriver: true
        })
      ]).start();
    });
  }, [orbOpacity, orbScale, diamondOpacity, transcriptionOpacity]);
  
  // Hold to talk — press starts recording, release sends audio
  const handleOrbPressIn = async () => {
    Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Heavy);
    setIsRecording(true);
    await AudioService.startRecording();
  };
  
  const handleOrbPressOut = async () => {
    setIsRecording(false);
    const audioBase64 = await AudioService.stopRecordingAndGetBase64();
    if (audioBase64) {
      SocketService.emit('mobile_voice_command', { audio: audioBase64 });
    }
  };
  
  return (
    <View style={styles.container}>
      <StatusBar barStyle="light-content" backgroundColor="#0a0a0f" />
      
      {/* Connection indicator top-right */}
      <View style={styles.connectionBadge}>
        <View style={[
          styles.connectionDot,
          { backgroundColor: isConnected ? COLORS.success : COLORS.error }
        ]} />
        <Text style={styles.connectionText}>
          {isConnected ? 'LINKED' : 'LOST'}
        </Text>
      </View>
      
      {/* SODA status label top-center */}
      <Text style={styles.statusLabel}>
        {sodaStatus.toUpperCase()}
      </Text>
      
      {/* Center area — either Orb OR Diamond Menu */}
      <View style={styles.centerArea}>
        
        {/* The Orb — animated, disappears when diamond shows */}
        <Animated.View style={[
          styles.orbWrapper,
          {
            opacity: orbOpacity,
            transform: [{ scale: orbScale }]
          }
        ]}>
          <SodaOrb
            status={sodaStatus}
            isRecording={isRecording}
            onPressIn={handleOrbPressIn}
            onPressOut={handleOrbPressOut}
          />
          <Text style={styles.orbHint}>
            {isRecording ? 'LISTENING...' : 'HOLD TO SPEAK'}
          </Text>
        </Animated.View>
        
        {/* Diamond Menu — appears when orb disappears */}
        {showDiamond && (
          <Animated.View style={[
            styles.diamondWrapper,
            { opacity: diamondOpacity }
          ]}>
            <DiamondMenu onClose={closeDiamondMenu} />
          </Animated.View>
        )}
        
      </View>
      
      {/* Transcription feed — bottom area */}
      <Animated.View style={[
        styles.transcriptionArea,
        { opacity: transcriptionOpacity }
      ]}>
        <TranscriptionFeed entries={transcriptions} />
      </Animated.View>
      
      {/* Manual options button — bottom right */}
      {!showDiamond && (
        <TouchableOpacity
          style={styles.optionsButton}
          onPress={triggerDiamondMenu}
        >
          <Text style={styles.optionsButtonText}>⊞</Text>
        </TouchableOpacity>
      )}
      
      {/* Scanlines — HUD decoration */}
      <View style={styles.scanlines} pointerEvents="none">
        {[...Array(6)].map((_, i) => (
          <View key={i} style={[styles.scanline, { top: `${15 * i}%` }]} />
        ))}
      </View>
      
    </View>
  );
}
Component: SodaOrb.js
This mirrors the HolographicOrb from the desktop but built for React Native using Animated and Canvas (via react-native-skia or pure Animated API):

JavaScript

// components/SodaOrb.js

/*
 The orb has these visual states that directly mirror desktop behavior:
 
 idle      → dim gray pulse, slow rhythm
 listening → bright cyan glow, faster pulse, rings expand on touch
 thinking  → amber/orange color, irregular pulse pattern  
 speaking  → full brightness, expanding outward rings
 
 Implementation strategy:
 - Core orb: a circular View with animated background gradient
 - Outer rings: 3 concentric Animated.View circles
 - Each ring has its own timing offset for organic feel
 - Glow: achieved with multiple layers of Views with decreasing opacity
   (React Native doesn't have CSS box-shadow glow, this approximates it)
 - Color interpolation: Animated.Value interpolated to color strings
*/

import React, { useEffect, useRef } from 'react';
import { View, TouchableWithoutFeedback, Animated, StyleSheet } from 'react-native';
import { LinearGradient } from 'expo-linear-gradient';
import { COLORS } from '../constants/colors';

// Color palettes per status — matches desktop HolographicOrb palettes
const STATUS_COLORS = {
  idle:      { core: '#1a1a2e', ring: '#2a2a4e', glow: '#00f0ff22' },
  listening: { core: '#003344', ring: '#00f0ff', glow: '#00f0ff44' },
  thinking:  { core: '#2d1a00', ring: '#ffaa00', glow: '#ffaa0044' },
  speaking:  { core: '#001a33', ring: '#00f0ff', glow: '#00f0ff66' },
};

export default function SodaOrb({ status, isRecording, onPressIn, onPressOut }) {
  // Animated values
  const coreScale = useRef(new Animated.Value(1)).current;
  const ring1Scale = useRef(new Animated.Value(1.2)).current;
  const ring2Scale = useRef(new Animated.Value(1.5)).current;
  const ring3Scale = useRef(new Animated.Value(1.8)).current;
  const glowOpacity = useRef(new Animated.Value(0.3)).current;
  const colorProgress = useRef(new Animated.Value(0)).current;
  
  // Refs to hold animation loop objects so we can stop them
  const coreAnim = useRef(null);
  const ringAnim = useRef(null);
  
  useEffect(() => {
    // Stop previous animations
    if (coreAnim.current) coreAnim.current.stop();
    if (ringAnim.current) ringAnim.current.stop();
    
    // Choose animation parameters based on status
    const config = {
      idle:      { speed: 2000, scale: 1.05, ringExpand: 0.15, glowMax: 0.3 },
      listening: { speed: 800,  scale: 1.12, ringExpand: 0.25, glowMax: 0.7 },
      thinking:  { speed: 500,  scale: 1.08, ringExpand: 0.2,  glowMax: 0.5 },
      speaking:  { speed: 300,  scale: 1.15, ringExpand: 0.3,  glowMax: 0.9 },
    }[status] || { speed: 2000, scale: 1.05, ringExpand: 0.15, glowMax: 0.3 };
    
    // Core pulse
    coreAnim.current = Animated.loop(
      Animated.sequence([
        Animated.timing(coreScale, {
          toValue: config.scale,
          duration: config.speed,
          useNativeDriver: true
        }),
        Animated.timing(coreScale, {
          toValue: 1,
          duration: config.speed,
          useNativeDriver: true
        })
      ])
    );
    coreAnim.current.start();
    
    // Staggered ring animations — each ring offset by 200ms
    ringAnim.current = Animated.loop(
      Animated.stagger(200, [

// components/DiamondMenu.js

import React, { useEffect, useRef } from 'react';
import {
  View, StyleSheet, Dimensions, Animated, Text, TouchableOpacity
} from 'react-native';
import * as Haptics from 'expo-haptics';
import OptionNode from './OptionNode';
import SocketService from '../services/SocketService';
import { COLORS } from '../constants/colors';
import { OPTIONS } from '../constants/options';

const { width, height } = Dimensions.get('window');

export default function DiamondMenu({ onClose }) {
  // Each option gets its own animated position + opacity + scale
  const nodeAnims = useRef(
    OPTIONS.map(() => ({
      x: new Animated.Value(0),          // starts at center
      y: new Animated.Value(0),          // starts at center
      opacity: new Animated.Value(0),
      scale: new Animated.Value(0.3),
    }))
  ).current;
  
  // Center diamond connector line opacity
  const lineOpacity = useRef(new Animated.Value(0)).current;
  const centerOpacity = useRef(new Animated.Value(0)).current;
  
  useEffect(() => {
    // Stagger the entrance animations — each node flies out 60ms after the previous
    // This creates the "explosion from center" feel
    const staggerDelay = 60;
    
    // First: flash the center point (50ms)
    Animated.timing(centerOpacity, {
      toValue: 1, duration: 50, useNativeDriver: true
    }).start();
    
    // Then: fly each node to its position
    const animations = OPTIONS.map((option, i) => {
      return Animated.delay(
        i * staggerDelay,
        Animated.parallel([
          // Move to final X position
          Animated.spring(nodeAnims[i].x, {
            toValue: option.position.x,
            speed: 14,
            bounciness: 8,
            useNativeDriver: true
          }),
          // Move to final Y position
          Animated.spring(nodeAnims[i].y, {
            toValue: option.position.y,
            speed: 14,
            bounciness: 8,
            useNativeDriver: true
          }),
          // Fade in
          Animated.timing(nodeAnims[i].opacity, {
            toValue: 1, duration: 200, useNativeDriver: true
          }),
          // Scale up from 0.3 to 1
          Animated.spring(nodeAnims[i].scale, {
            toValue: 1, speed: 16, bounciness: 12,
            useNativeDriver: true
          })
        ])
      );
    });
    
    // Run all simultaneously (each has its own delay)
    Animated.parallel(animations).start();
    
    // Connector lines fade in after nodes are out (300ms delay)
    setTimeout(() => {
      Animated.timing(lineOpacity, {
        toValue: 1, duration: 400, useNativeDriver: true
      }).start();
    }, 300);
    
  }, []);
  
  // Collapse animation — reverse of entrance, all nodes snap back to center
  const handleClose = () => {
    Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light);
    
    // Snap everything back fast
    const collapseAnims = OPTIONS.map((option, i) =>
      Animated.parallel([
        Animated.spring(nodeAnims[i].x, {
          toValue: 0, speed: 30, bounciness: 0, useNativeDriver: true
        }),
        Animated.spring(nodeAnims[i].y, {
          toValue: 0, speed: 30, bounciness: 0, useNativeDriver: true
        }),
        Animated.timing(nodeAnims[i].opacity, {
          toValue: 0, duration: 150, useNativeDriver: true
        }),
        Animated.timing(nodeAnims[i].scale, {
          toValue: 0.3, duration: 150, useNativeDriver: true
        })
      ])
    );
    
    Animated.parallel([
      ...collapseAnims,
      Animated.timing(lineOpacity, {
        toValue: 0, duration: 100, useNativeDriver: true
      })
    ]).start(() => onClose());
  };
  
  const handleOptionPress = (option) => {
    Haptics.notificationAsync(Haptics.NotificationFeedbackType.Success);
    
    // Navigate to the specific option's sub-screen
    // OR send a direct socket command for simple actions
    switch(option.action) {
      case 'screenshot':
        SocketService.emit('mobile_text_command', {
          text: 'take a screenshot and show me'
        });
        break;
      case 'volume':
        // Navigate to volume control sub-screen
        option.onNavigate();
        break;
      case 'mouse':
        option.onNavigate();
        break;
      // ... etc
    }
    
    handleClose();
  };
  
  return (
    <View style={styles.diamondContainer}>
      
      {/* Center point — pulsing dot where orb was */}
      <Animated.View style={[
        styles.centerPoint,
        { opacity: centerOpacity }
      ]} />
      
      {/* Connector lines — subtle lines from center to each node */}
      <Animated.View style={[styles.linesLayer, { opacity: lineOpacity }]}>
        {OPTIONS.map((option, i) => (
          <ConnectorLine key={i} toX={option.position.x} toY={option.position.y} />
        ))}
      </Animated.View>
      
      {/* The 8 option nodes */}
      {OPTIONS.map((option, i) => (
        <Animated.View
          key={option.id}
          style={[
            styles.nodeWrapper,
            {
              opacity: nodeAnims[i].opacity,
              transform: [
                { translateX: nodeAnims[i].x },
                { translateY: nodeAnims[i].y },
                { scale: nodeAnims[i].scale }
              ]
            }
          ]}
        >
          <OptionNode
            option={option}
            onPress={() => handleOptionPress(option)}
          />
        </Animated.View>
      ))}
      
      {/* Close button — top right corner */}
      <TouchableOpacity style={styles.closeButton} onPress={handleClose}>
        <Text style={styles.closeText}>✕</Text>
      </TouchableOpacity>
      
    </View>
  );
}

services/AudioService.js — Voice In Both Directions

// services/AudioService.js

/*
 Handles two directions of audio:
 
 1. YOUR VOICE → PC (recording)
    - Record mic using expo-av Audio.Recording
    - Convert to PCM base64
    - Send via Socket.IO mobile_voice_command
 
 2. SODA'S VOICE → YOUR PHONE (playback)  
    - Receive PCM base64 from soda_audio socket event
    - Decode to audio buffer
    - Play through expo-av Audio.Sound
    - Queue multiple chunks for smooth playback
*/

import { Audio } from 'expo-av';

class AudioService {
  constructor() {
    this.recording = null;
    this.soundQueue = [];
    this.isPlaying = false;
  }
  
  async initialize() {
    // Request mic permission
    const { granted } = await Audio.requestPermissionsAsync();
    if (!granted) throw new Error('Microphone permission denied');
    
    // Set audio mode: allow recording + speaker playback
    await Audio.setAudioModeAsync({
      allowsRecordingIOS: true,
      playsInSilentModeIOS: true,
      shouldDuckAndroid: true,
    });
  }
  
  async startRecording() {
    if (this.recording) return;
    
    // Recording settings that match SODA's 24kHz PCM format
    const recordingOptions = {
      android: {
        extension: '.wav',
        outputFormat: Audio.RECORDING_OPTION_ANDROID_OUTPUT_FORMAT_DEFAULT,
        audioEncoder: Audio.RECORDING_OPTION_ANDROID_AUDIO_ENCODER_DEFAULT,
        sampleRate: 24000,
        numberOfChannels: 1,
        bitRate: 128000,
      },
      ios: {
        extension: '.wav',
        audioQuality: Audio.RECORDING_OPTION_IOS_AUDIO_QUALITY_HIGH,
        sampleRate: 24000,
        numberOfChannels: 1,
        bitRate: 128000,
        linearPCMBitDepth: 16,
        linearPCMIsBigEndian: false,
        linearPCMIsFloat: false,
      }
    };
    
    const { recording } = await Audio.Recording.createAsync(recordingOptions);
    this.recording = recording;
  }
  
  async stopRecordingAndGetBase64() {
    if (!this.recording) return null;
    
    await this.recording.stopAndUnloadAsync();
    const uri = this.recording.getURI();
    this.recording = null;
    
    // Read file and convert to base64
    const response = await fetch(uri);
    const blob = await response.blob();
    
    return new Promise((resolve) => {
      const reader = new FileReader();
      reader.onloadend = () => {
        // Strip the data:audio/wav;base64, prefix
        const base64 = reader.result.split(',')[1];
        resolve(base64);
      };
      reader.readAsDataURL(blob);
    });
  }
  
  async playPcmBase64(base64Audio) {
    /*
     SODA sends 24kHz 16-bit PCM audio.
     expo-av can play WAV files.
     We create a temporary WAV file from the raw PCM bytes.
     A WAV file is just a 44-byte header + raw PCM data.
    */
    const binaryString = atob(base64Audio);
    const bytes = new Uint8Array(binaryString.length);
    for (let i = 0; i < binaryString.length; i++) {
      bytes[i] = binaryString.charCodeAt(i);
    }
    
    // Build WAV header for 24kHz, mono, 16-bit PCM
    const header = buildWavHeader(bytes.length, 24000, 1, 16);
    const wavData = new Uint8Array(header.length + bytes.length);
    wavData.set(header);
    wavData.set(bytes, header.length);
    
    // Convert to base64 WAV URI that expo-av can play
    const base64Wav = btoa(String.fromCharCode(...wavData));
    const uri = `data:audio/wav;base64,${base64Wav}`;
    
    const { sound } = await Audio.Sound.createAsync({ uri });
    await sound.playAsync();
    
    // Clean up after playback
    sound.setOnPlaybackStatusUpdate((status) => {
      if (status.didJustFinish) {
        sound.unloadAsync();
      }
    });
  }
  
  stopAll() {
    if (this.recording) {
      this.recording.stopAndUnloadAsync();
      this.recording = null;
    }
  }
}

// Build a minimal WAV header
function buildWavHeader(dataSize, sampleRate, channels, bitsPerSample) {
  const header = new ArrayBuffer(44);
  const view = new DataView(header);
  const byteRate = sampleRate * channels * bitsPerSample / 8;
  const blockAlign = channels * bitsPerSample / 8;
  
  // RIFF chunk
  view.setUint32(0, 0x52494646, false);  // 'RIFF'
  view.setUint32(4, 36 + dataSize, true); // file size - 8
  view.setUint32(8, 0x57415645, false);   // 'WAVE'
  // fmt chunk
  view.setUint32(12, 0x666d7420, false);  // 'fmt '
  view.setUint32(16, 16, true);           // chunk size
  view.setUint16(20, 1, true);            // PCM format
  view.setUint16(22, channels, true);
  view.setUint32(24, sampleRate, true);
  view.setUint32(28, byteRate, true);
  view.setUint16(32, blockAlign, true);
  view.setUint16(34, bitsPerSample, true);
  // data chunk
  view.setUint32(36, 0x64617461, false);  // 'data'
  view.setUint32(40, dataSize, true);
  
  return new Uint8Array(header);
}

export default new AudioService();
Additional Features You Should Add
Beyond what you described, here are features that make this significantly more powerful:

1. Live Screen Mirror
Stream your PC screen to your phone at 5fps. You can see exactly what's on your PC. Uses the existing screen_vision.py capture but sends frames to mobile.

2. Remote Touchpad
Your phone screen becomes a trackpad. Touch position mapped to PC mouse position. Tap = left click. Two-finger tap = right click. Two-finger scroll = scroll wheel. This uses screen_control.py's mouse functions already built in SODA.

3. Clipboard Sync
Copy something on your phone → it appears on your PC clipboard. Copy on PC → your phone receives it. Uses system_local.py's clipboard tools.

4. Quick Command Shortcuts
Swipe up on the orb → reveal 3 voice shortcut buttons for your most-used commands. These are stored locally and configurable.

5. Notification Mirror
When SODA fires a reminder, your phone buzzes. When a tool finishes running, you get a push notification. Uses expo-notifications.

6. Volume & Media Remote
Dedicated volume slider screen. Previous/Next/Play-Pause media keys. These map directly to SODA's system_control.py volume functions.

7. Wake-on-LAN
If your PC is asleep, button in app sends a magic packet to wake it before connecting.
