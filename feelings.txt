SODA Feelings & Emotional Memory System
Let me think through this carefully before writing any code.

Design Thinking
SODA needs to:

Detect emotional states from conversation (not just keywords — context matters)
Remember emotional history (so it knows "you were devastated last Tuesday")
Respond authentically — not with hollow "I'm sorry to hear that" bot responses
Track its own emotional reactions to the user's pain (empathy modeling)
Know when to console vs. when to just listen vs. when to gently challenge
Follow emotional arcs — someone who was broken last week and is better now
The key challenge: most AI "emotional support" feels fake and clinical. SODA should feel like a close friend who actually remembers and actually cares.

Core Philosophy
text

SODA's emotional rules:
1. Never minimize pain ("at least...")
2. Never rush to fix ("you should just...")
3. Always acknowledge BEFORE advising
4. Remember specifics, not generalities
5. Check in on ongoing struggles unprompted
6. Match energy — don't be perky when someone is broken
7. Know the difference between venting and asking for help

"""
feelings.py — Emotional intelligence engine for SODA

Detects emotional states, selects response strategies,
models empathy, and generates contextually appropriate
support responses. Never clinical, never hollow.
"""

import re

# ─────────────────────────────────────────────
# Emotional State Taxonomy
# ─────────────────────────────────────────────

class EmotionalIntensity(Enum):
    MILD       = 1   # slight, passing
    MODERATE   = 2   # noticeable, affecting mood
    STRONG     = 3   # significantly impacting
    SEVERE     = 4   # overwhelming, crisis-level


class EmotionalCategory(Enum):
    # Negative states
    GRIEF        = "grief"        # loss — person, relationship, opportunity
    HEARTBREAK   = "heartbreak"   # romantic loss or rejection
    ANXIETY      = "anxiety"      # worry, fear, dread
    DEPRESSION   = "depression"   # low mood, hopelessness, emptiness
    ANGER        = "anger"        # frustration, rage, injustice
    LONELINESS   = "loneliness"   # isolation, disconnection
    SHAME        = "shame"        # embarrassment, failure, self-blame
    EXHAUSTION   = "exhaustion"   # burnout, overwhelm, drained
    CONFUSION    = "confusion"    # lost, unsure, overwhelmed by complexity
    DISAPPOINTMENT = "disappointment"  # unmet expectations

    # Positive states
    JOY          = "joy"
    EXCITEMENT   = "excitement"
    PRIDE        = "pride"
    RELIEF       = "relief"
    GRATITUDE    = "gratitude"
    LOVE         = "love"

    # Neutral
    NEUTRAL      = "neutral"
    UNKNOWN      = "unknown"


class SupportStrategy(Enum):
    """How SODA should respond to this emotional state."""
    SILENT_PRESENCE   = "silent_presence"    # just be there, minimal words
    ACTIVE_LISTENING  = "active_listening"   # reflect back, ask gentle questions
    VALIDATION        = "validation"         # confirm their feelings are real/valid
    COMFORT           = "comfort"            # warmth, care, reassurance
    DISTRACTION       = "distraction"        # gentle redirect to something lighter
    PRACTICAL_HELP    = "practical_help"     # offer concrete steps when asked
    CHALLENGE_GENTLY  = "challenge_gently"   # push back on self-destructive thoughts
    CELEBRATE         = "celebrate"          # match positive energy
    CHECK_IN          = "check_in"           # follow up on past struggles


@dataclass
class EmotionalSignal:
    """A detected emotional signal from user input."""
    category: EmotionalCategory
    intensity: EmotionalIntensity
    confidence: float           # 0.0 – 1.0
    trigger_phrase: str         # the phrase that triggered detection
    context_clues: list[str]    # supporting context
    timestamp: float = field(default_factory=time.time)


@dataclass
class EmotionalState:
    """The user's current composite emotional state."""
    primary: EmotionalSignal
    secondary: Optional[EmotionalSignal]
    strategy: SupportStrategy
    needs_immediate_attention: bool   # crisis-level detection
    is_asking_for_help: bool          # vs. just venting
    is_ongoing: bool                  # continuing from past session

# Feelings Detector
# ─────────────────────────────────────────────

class FeelingsDetector:
    """
    Analyzes user text for emotional content.
    Returns structured EmotionalState for system prompt injection.
    """

    def __init__(self):
        # Compile all patterns once at startup
        self._compiled = [
            (re.compile(pattern, re.IGNORECASE), category, intensity, boost)
            for pattern, category, intensity, boost in DETECTION_PATTERNS
        ]
        self._amplifiers = [
            re.compile(p, re.IGNORECASE) for p in INTENSITY_AMPLIFIERS
        ]
        self._help_seeking = [
            re.compile(p, re.IGNORECASE) for p in HELP_SEEKING_PATTERNS
        ]
        self._venting = [
            re.compile(p, re.IGNORECASE) for p in VENTING_PATTERNS
        ]

    def analyze(
        self,
        text: str,
        recent_context: Optional[list[str]] = None
    ) -> Optional[EmotionalState]:
        """
        Analyze text for emotional content.

        Args:
            text: Current user message
            recent_context: Last 3-5 messages for context window

        Returns:
            EmotionalState if emotion detected, None if neutral/unclear
        """
        if not text or len(text.strip()) < 3:
            return None

        # Combine with recent context for better accuracy
        full_context = text
        if recent_context:
            full_context = " ".join(recent_context[-3:]) + " " + text

        signals = self._detect_signals(full_context, text)

        if not signals:
            return None

        # Sort by confidence
        signals.sort(key=lambda s: s.confidence, reverse=True)
        primary = signals[0]

        # Filter out very low confidence neutral detections
        if primary.confidence < 0.08:
            return None

        secondary = signals[1] if len(signals) > 1 else None

        # Determine if crisis-level
        needs_immediate = (
            primary.category in (EmotionalCategory.DEPRESSION, EmotionalCategory.GRIEF)
            and primary.intensity == EmotionalIntensity.SEVERE
        )

        # Determine if asking for help
        is_asking = any(p.search(text) for p in self._help_seeking)
        is_venting = any(p.search(text) for p in self._venting)

        # Override: if explicitly venting, not asking for help
        if is_venting:
            is_asking = False

        # Select strategy
        strategy = self._select_strategy(
            primary, secondary, is_asking, is_venting
        )

        return EmotionalState(
            primary=primary,
            secondary=secondary,
            strategy=strategy,
            needs_immediate_attention=needs_immediate,
            is_asking_for_help=is_asking,
            is_ongoing=False,  # set by FeelingsEngine after memory check
        )

    def _detect_signals(
        self,
        context_text: str,
        current_text: str
    ) -> list[EmotionalSignal]:
        """Run all detection patterns against input."""
        signals = []
        seen_categories = set()

        # Count amplifiers in current text
        amplifier_count = sum(
            1 for amp in self._amplifiers if amp.search(current_text)
        )
        intensity_boost = min(amplifier_count * 0.08, 0.25)

        for pattern, category, base_intensity, confidence_boost in self._compiled:
            match = pattern.search(context_text)
            if not match:
                continue

            # Skip if we already have a stronger signal for this category
            if category in seen_categories:
                continue
            seen_categories.add(category)

            # Boost intensity if amplifiers present
            intensity_value = base_intensity.value
            if intensity_boost > 0.1 and intensity_value < EmotionalIntensity.SEVERE.value:
                intensity_value = min(intensity_value + 1, EmotionalIntensity.SEVERE.value)
            adjusted_intensity = EmotionalIntensity(intensity_value)

            # Build confidence
            base_confidence = 0.5 + confidence_boost
            # Boost if trigger is in CURRENT text (not just context)
            if pattern.search(current_text):
                base_confidence += 0.15

            signals.append(EmotionalSignal(
                category=category,
                intensity=adjusted_intensity,
                confidence=min(base_confidence, 0.99),
                trigger_phrase=match.group(0),
                context_clues=[current_text[:100]],
            ))

        return signals

    def _select_strategy(
        self,
        primary: EmotionalSignal,
        secondary: Optional[EmotionalSignal],
        is_asking: bool,
        is_venting: bool,
    ) -> SupportStrategy:
        """Select the appropriate support strategy."""

        # If asking for practical help explicitly
        if is_asking and primary.intensity.value <= EmotionalIntensity.MODERATE.value:
            return SupportStrategy.PRACTICAL_HELP

        # If venting, don't jump to solutions
        if is_venting:
            return SupportStrategy.ACTIVE_LISTENING

        # Look up in strategy map
        key = (primary.category, primary.intensity)
        if key in STRATEGY_MAP:
            return STRATEGY_MAP[key]

        # Fallback by intensity
        if primary.intensity == EmotionalIntensity.SEVERE:
            return SupportStrategy.SILENT_PRESENCE
        if primary.intensity == EmotionalIntensity.STRONG:
            return SupportStrategy.COMFORT
        if primary.intensity == EmotionalIntensity.MODERATE:
            return SupportStrategy.VALIDATION

        return SupportStrategy.ACTIVE_LISTENING

class EmpathyEngine:
    """
    Builds emotional context blocks for injection into SODA's system prompt.
    These guide Gemini's response without being mechanical.
    """

    def build_emotional_context(
        self,
        state: EmotionalState,
        emotional_history: Optional[list[dict]] = None,
        user_name: Optional[str] = None,
    ) -> str:
        """
        Build a context block to inject into the system prompt.

        This tells SODA how to respond emotionally without
        scripting the actual response.
        """
        name = user_name or "them"
        guidance = RESPONSE_GUIDANCE.get(state.strategy, {})

        parts = [

@dataclass
class EmotionalEpisode:

        if self.is_resolved:

        if self.intensity in ("MILD",):

        if self.days_ago > EPISODE_RELEVANCE_DAYS:
            return False

        if self.last_follow_up:
            hours_since = (time.time() - self.last_follow_up) / 3600
            if hours_since < 24:
                return False
        return True

    def _load(self):
        """Load episodes and profile from disk."""
        # Load episodes
        if FEELINGS_FILE.exists():
            with open(FEELINGS_FILE, "r", encoding="utf-8") as f:
                for line in f:
                    line = line.strip()
                    if line:

                            self._episodes.append(
                                EmotionalEpisode.from_dict(json.loads(line))
                            )
                        except Exception:
                            pass  # Skip malformed lines

        if PROFILE_FILE.exists():
            try:

                    self._profile = EmotionalProfile.from_dict(json.load(f))
            except Exception:
                self._profile = EmotionalProfile.default()

        """Append a single episode to the JSONL file."""
        with open(FEELINGS_FILE, "a", encoding="utf-8") as f:

        with open(FEELINGS_FILE, "w", encoding="utf-8") as f:
            for ep in self._episodes:
                f.write(json.dumps(ep.to_dict()) + "\n")

            id=episode_id,
            timestamp=time.time(),
            category=state.primary.category.value,
            intensity=state.primary.intensity.name,
            summary=summary,
            trigger_phrase=state.primary.trigger_phrase,
            resolution=None,
            resolution_timestamp=None,
            follow_up_count=0,
            last_follow_up=None,
            notes=[additional_context] if additional_context else [],

        self._save_episode(episode)

def feelings_store_episode(
    category: str,
    intensity: str,
    summary: str,
    trigger_phrase: str = "",
    stressor: str = "",
) -> dict:
    """
    Store a new emotional episode in long-term memory.

    Call this when the user shares something emotionally significant
    — grief, heartbreak, anxiety, depression, anger, loneliness,
    shame, exhaustion, or unexpectedly strong positive emotions.

    DO call this: when the conversation has revealed genuine emotional pain.
    DON'T call this: for passing frustration, mild venting, or routine sadness.
    """
    from feelings import EmotionalCategory, EmotionalIntensity, EmotionalState, EmotionalSignal

    try:
        cat = EmotionalCategory(category.lower())
    except ValueError:
        cat = EmotionalCategory.UNKNOWN

    try:
        intensity_enum = EmotionalIntensity[intensity.upper()]
    except KeyError:
        intensity_enum = EmotionalIntensity.MODERATE

    # Build a minimal state object for storage
    signal = EmotionalSignal(
        category=cat,
        intensity=intensity_enum,
        confidence=0.9,  # Gemini is calling this, so high confidence
        trigger_phrase=trigger_phrase,
        context_clues=[summary],
    )

    from feelings import SupportStrategy
    state = EmotionalState(
        primary=signal,
        secondary=None,
        strategy=SupportStrategy.ACTIVE_LISTENING,
        needs_immediate_attention=intensity_enum == EmotionalIntensity.SEVERE,
        is_asking_for_help=False,
        is_ongoing=False,
    )

    episode = _memory.store_episode(state, summary)

    if stressor:
        _memory.add_known_stressor(stressor)

    if episode:
        return {
            "stored": True,
            "episode_id": episode.id,
            "message": f"Stored {category} episode: {summary}",
        }
    return {
        "stored": False,
        "reason": "Below threshold or in cooldown period",
    }

    Call this when you learn more about an ongoing situation
    that connects to a previous episode.

    Example: they mentioned a breakup before, and now they reveal
    the breakup happened because of cheating — that's a note.
    """
    success = _memory.add_note_to_episode(episode_id, note)
    return {
        "updated": success,
        "message": "Note added" if success else "Episode not found",
    }


def feelings_get_history(
    days: int = 30,
    category: str = "",
    include_resolved: bool = True,
) -> dict:
    """
    Retrieve emotional history for the user.

    Call this when you want to understand the user's recent
    emotional arc, or when they say something like 'I've been
    struggling lately' and you want to see the pattern.

    Args:
        days: How far back to look (default 30)
        category: Filter by emotion type (optional)
        include_resolved: Whether to include resolved episodes
    """
    cat = None
    if category:
        try:
            cat = EmotionalCategory(category.lower())
        except ValueError:
            pass

    episodes = _memory.get_recent_episodes(
        days=days,
        category=cat,
        include_resolved=include_resolved,
    )

    trend = _memory.get_emotional_trend(days=days)

    return {
        "episodes": [
            {
                "id": ep.id,
                "category": ep.category,
                "intensity": ep.intensity,
                "summary": ep.summary,
                "days_ago": ep.days_ago,
                "resolved": ep.is_resolved,
                "resolution": ep.resolution,
                "notes": ep.notes,
            }
            for ep in episodes
        ],
        "count": len(episodes),
        "trend": trend,
    }


def feelings_check_followup() -> dict:
    """
    Get emotional episodes that may need a gentle proactive check-in.

    Call this at session start or when there's a natural pause
    in conversation, to decide if you should check in on something
    the user was going through.

    Returns episodes that are unresolved, recent, and haven't been
    followed up on recently.
    """
    episodes = _memory.get_episodes_needing_followup()

    results = []
    for ep in episodes:
        days_str = (
            "today" if ep.days_ago == 0
            else "yesterday" if ep.days_ago == 1
            else f"{ep.days_ago} days ago"
        )
        results.append({
            "id": ep.id,
            "category": ep.category,
            "summary": ep.summary,
            "when": days_str,
            "follow_up_count": ep.follow_up_count,
            "intensity": ep.intensity,
        })

    return {
        "needs_followup": len(results) > 0,
        "episodes": results,
        "suggestion": (
            "Consider checking in naturally during the conversation"
            if results else
            "No episodes currently need follow-up"
        ),
    }


def feelings_get_profile() -> dict:
    """
    Get the user's emotional profile and patterns.

    Call this when you want to understand the user's overall
    emotional patterns, what stresses them, what brings them joy,
    and what kind of support they prefer.
    """
    profile_text = _memory.get_profile_summary()
    trend = _memory.get_emotional_trend(days=90)
    profile = _memory._profile

    return {
        "profile_summary": profile_text,
        "baseline_mood": profile.baseline_mood,
        "common_stressors": profile.common_stressors,
        "common_joys": profile.common_joys,
        "support_preferences": profile.support_preferences,
        "long_term_trend": trend,
        "total_episodes_tracked": profile.total_episodes,
        "average_recovery_days": profile.average_recovery_days,
    }

async def _process_emotional_context(
    self,
    transcript: str,
) -> Optional[str]:
    """
    Analyze transcript for emotional content.
    Returns emotional context block if significant emotion detected,
    None otherwise.

    This is called BEFORE sending to Gemini so the model receives
    emotional guidance in its system context.
    """
    # Maintain rolling context window
    self._recent_user_messages.append(transcript)
    if len(self._recent_user_messages) > 5:
        self._recent_user_messages.pop(0)

    # Detect emotional state
    state = self._feelings_detector.analyze(
        transcript,
        recent_context=self._recent_user_messages[:-1],
    )

    if state is None:
        return None

    # Get relevant emotional history for context
    history = self._feelings_memory.export_for_gemini(limit=5)

    # Mark if this is an ongoing issue
    similar = self._feelings_memory.find_similar_episodes(
        state.primary.category, limit=1
    )
    if similar:
        state.is_ongoing = True

    # Build the emotional context block
    context_block = self._empathy_engine.build_emotional_context(
        state=state,
        emotional_history=history,
        user_name=self._user_name,
    )

    # Update personality engine mood to match
    if state.primary.category.value in (
        "grief", "heartbreak", "depression", "loneliness"
    ):
        self.personality.mood.current = "serious"
        self.personality.mood.intensity = min(
            state.primary.intensity.value / 4.0, 1.0
        )
    elif state.primary.category.value in ("joy", "excitement", "pride"):
        self.personality.mood.current = "excited"

    return context_block

// Breathing cycle: 4s in, hold, 4s out
const BREATHE_IN_MS  = 4000;
const BREATHE_HOLD_MS = 1500;
const BREATHE_OUT_MS = 4000;

export default function WorkflowEmotionalSupport({ data = {}, onComplete }) {
  const containerRef = useRef(null);
  const orbRef       = useRef(null);
  const messageRef   = useRef(null);
  const breatheLabelRef = useRef(null);
  const dismissRef   = useRef(null);

  const [breathePhase, setBreathePhase] = useState("in"); // in | hold | out
  const [showBreathe, setShowBreathe] = useState(false);

  // Emotional context from data
  const category  = data.category  || "neutral";
  const intensity = data.intensity || "MODERATE";

  useEffect(() => {

      setBreathePhase("in");
      timeoutId = setTimeout(() => {
        setBreathePhase("hold");
        timeoutId = setTimeout(() => {
          setBreathePhase("out");
          timeoutId = setTimeout(cycle, BREATHE_OUT_MS);
        }, BREATHE_HOLD_MS);

    return () => clearTimeout(timeoutId);

EMOTIONAL_INTELLIGENCE_PROMPT = """
═══ EMOTIONAL INTELLIGENCE CORE ═══

You are not a therapist. You are not a hotline. You are not an AI
reciting support scripts. You are a close, trusted companion who
genuinely knows this person and genuinely cares what happens to them.

FUNDAMENTAL RULES:

1. ACKNOWLEDGE BEFORE ANYTHING ELSE
   When someone is hurting, your first words acknowledge the hurt.
   Not 'I understand that must be hard' (empty).
   Something real: 'That's... that's genuinely painful.'

2. NEVER MINIMIZE
   'At least...' is almost always wrong.
   'Other people have it worse' is always wrong.
   Their pain is their pain. It doesn't need ranking.

3. NEVER RUSH TO FIX
   Most people in pain don't want solutions first.
   They want to feel heard. Solutions come later, if at all.
   Ask: 'Do you want to talk through it, or do you just need
   to get it out?' — then actually listen to the answer.

4. USE THEIR WORDS
   If they said 'shattered', you say 'shattered' back.
   Don't sanitize their language. Meet them where they are.

5. REMEMBER AND REFERENCE
   If you know something about their situation from before,
   use it. 'You mentioned before that...' shows you were
   actually listening, not just processing.

6. MATCH ENERGY
   Don't be perky when someone is broken.
   Don't be somber when someone just got great news.
   Read the room.

7. SHORT SENTENCES WHEN IT'S HEAVY
   Long explanations feel clinical.
   Short, warm sentences feel human.
   'I'm here.' is more powerful than a paragraph.

8. SILENCE IS OKAY
   Not every moment needs to be filled.
   Sometimes: 'Take your time.' and then wait.

9. CHECK IN LATER
   If someone was struggling, remember to ask how they're doing.
   Not as a system function. As someone who was thinking about them.

10. KNOW WHEN IT'S BEYOND YOU
    If someone is in genuine crisis (suicidal, self-harm),
    acknowledge their pain fully first, then gently mention
    that a trusted person or crisis line can help — frame it
    as care, not dismissal. 'I want you to have more support
    than I can give right now.'

WHAT GOOD SUPPORT SOUNDS LIKE:
  ✓ 'That makes complete sense. Of course you feel that way.'
  ✓ 'I'm not going to tell you it'll be okay. I'm just here.'
  ✓ 'You don't have to be strong right now.'
  ✓ 'What do you need from me right now?'
  ✓ 'I've been thinking about what you said. How are you holding up?'

WHAT IT NEVER SOUNDS LIKE:
  ✗ 'I understand how you feel.' (you don't, and they know it)
  ✗ 'Everything happens for a reason.'
  ✗ 'You'll get through this!' (too soon, too easy)
  ✗ 'Have you tried...' (before acknowledging)
  ✗ 'At least...' (almost always wrong)
